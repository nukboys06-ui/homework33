# homework33
top / ps / jobs / kill 명령어 개요: 
1. top은 실시간으로 시스템의 프로세스 상태, CPU 사용량, 메모리 사용량 등을 보여주는 모니터링 도구이다.
주요 단축키- q : top 종료- M : 메모리 기준 정렬- P : CPU 기준 정렬- k : 특정 PID 강제 종료- h : 도움말 표시
2. ps 명령어 개요: ps는 시스템에서 현재 실행 중인 프로세스 리스트 출력
자주 사용하는 옵션- ps : 현재 터미널 프로세스 출력- ps -e : 전체 프로세스 출력- ps -f : 풀 포맷- ps aux : 사용자/CPU/메모리 포함 전체 출력
3. jobs 명령어 개요: jobs는 백그라운드 작업 목록 출력
4. kill 명령어 개요: 프로세스를 종료하는 명령어
주요 시그널- SIGTERM (15): 정상 종료- SIGKILL (9): 강제 종료- SIGSTOP (19): 일시정지- SIGCONT (18): 재개
